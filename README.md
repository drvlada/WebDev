# HealthPlate

Виконали: Драган Влада, Завальська Анастасія КІ-33

## Опис проєкту
HealthPlate - це онлайн-сервіс для персоналізованого планування здорового харчування. Користувачі можуть створювати індивідуальні плани харчування, враховуючи вік, стать, рівень активності, цілі, харчові вподобання та алергії. Сервіс надаватиме готові рецепти, підрахунок калорій та формування списку продуктів.

## Основний функціонал 
* Можливість зареєструватись, увійти в профіль, а також редагувати його
* Перегляд стрічки зі статтями
* Пошук рецептів за фільтрами, перегляд їх харчової цінності, зберігання улюблених рецептів
* Створення персоналізованого меню на основі особистих параметрів та мети (зменшення/утримання/набір ваги)
* Можливість зворотнього зв'язку

## Основні сторінки
* index.html - головна сторінка, опис функціоналу сервісу
* about.html - інформація про розробників
* authorization.html - авторизація користувача
* community.html - стрічка, у якій публікуються статті пов'язані зі здоров'ям та їжею
* contacts.html - сторінка з контактами для зворотного зв'язку
* menu.html - створення персоналізованого меню
* profile.html - профіль користувача
* recipes.html - перелік рецептів за вподобаннями

## Опис використаних технологій
HTML5

Використано для створення семантичної та логічно структурованої розмітки сторінок.

CSS3

Застосовано для оформлення інтерфейсу, побудови макетів, виконання адаптивної верстки та організації структури стилів.

JavaScript

Мінімальне використання для реалізації інтерактивних елементів, зокрема бургер-меню.

Git

Використано як систему контролю версій для збереження історії змін.

Використані HTML-елементи та теги
Семантичні елементи

<header>, <nav>, <main>, <footer> — основні структурні блоки сторінок.

<section> — логічне групування контенту.

<article> — окремі інформаційні секції.

<form> — форми реєстрації та авторизації.

<figure>, <figcaption> — оформлення медіа-елементів.

Текстові елементи

<h1>, <h2>, <h3> — заголовки різного рівня.

<p> — текстові абзаци.

<ul>, <li> — марковані списки.

<strong>, <span> — акцентування та виділення тексту.

Форми та введення

<input> — текстові поля, email, пароль.

<label> — підписи до полів.

<button> — кнопки надсилання та підтвердження.

<textarea> — багаторядкові поля.

Медіа та навігація

<img> — зображення з атрибутом alt.

<a> — внутрішні й зовнішні посилання.

Службові елементи

<div> — універсальні контейнерні блоки.

<hr> — візуальне розділення секцій.

Використані технології CSS
Базові можливості

CSS Custom Properties (змінні) — централізоване управління кольорами, відступами та параметрами дизайну.

Flexbox — побудова гнучких контейнерів для навігації, блоків контенту та елементів інтерфейсу.

CSS Grid — створення адаптивних сіток для розміщення карток і секцій.

Media Queries — адаптивне налаштування розмітки під різні ширини екранів.

Reset-стилі через конструкцію

*,
*::before,
*::after {
  box-sizing: border-box;
}

Гнучкі сітки, що автоматично змінюють кількість елементів у рядку.
